# 🛒 Products App (FakeStore API)

A modern React-based product listing application that fetches data from an external API and displays products in a grid with a detailed view.

## 📌 Features

* 🛍️ View all products in a responsive grid
* 🔍 Click on a product to see detailed information
* ⏳ Loading state while fetching data
* 📱 Clean and user-friendly UI
* 🌐 API integration using FakeStore API

---

## 🔗 API Used

* FakeStore API
  👉 https://fakestoreapi.com/products

---

## 💡 Functionalities

### 1. Products Grid

* Displays all products with:

  * Image
  * Title
  * Price

### 2. Product Details View

* Shows:

  * Product Image
  * Title
  * Description
  * Price
  * Category

### 3. Navigation

* Click on product → Opens details
* Click **Back** → Returns to product list

## ⚙️ Installation & Setup

1. Clone the repository:

```
git clone <your-repo-link>
```

2. Navigate to project folder:

```
cd products-app
```

3. Install dependencies:

```
npm install
```

4. Start the application:

```
npm start
```

---

## 🌐 Deployment

### Vercel

1. Build the project:

```
npm run build
```

2. Upload the `build` folder to Netlify
